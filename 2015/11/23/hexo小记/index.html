hexo小记

严重的拖延症致使这个blog搭了超过半年才搭成 (๑′ᴗ‵๑)…
写点东西记录一下 第一次使用Markdown

Git安装

  1. mac安装Git
    $ brew install git
  2. 安装Node.js
    $ brew install node

配置SSH keys

  1. 生成SSH key
    $ ssh-keygen -t rsa -C "****@xx.com" #注册github的邮箱地址
  2. 输入密码验证
  3. 添加SSH到Github
    -找到并打开.ssh\id_rsa.pub文件，需要先设置显示隐藏文件
    -登陆github.Settings—>SSH keys—> add SSH keys
    -把之前密钥复制到key文本框中
  4. 测试是否成功
    $ ssh -T git@github.com
  5. 设置用户信息
    $ git config --global user.name "用户名"
    $ git config --global user.email "邮箱"

hexo安装设置

  1. 安装hexo
    $ npm install hexo -g
  2. init
    $ hexo init <folder>
    $ hexo g #generate 生成静态网页
    $ hexo s #运行本地服务
    $ hexo deploy #部署到github

    更改主题

    记得在./_config.yml修改主题名称

常用命令

$ hexo new "文章名"
生成文件source/_posts\文章名.md

每次部署过程

$ hexo clean
$ hexo g
$ hexo d
